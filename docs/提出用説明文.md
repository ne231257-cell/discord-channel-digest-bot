# 提出用説明文

## 作品名

Channel Digest Bot

## 作品の概要

Discordチャンネル内の直近10〜50件の会話を取得し、Gemini APIを使って「会話の概要」「決定事項」「タスク」「未解決の質問」「次の一歩」に整理するチャットボットです。大学のグループ課題やチーム制作で、流れてしまった会話を短時間で確認するために利用できます。

## Discordで実現する意味

ユーザーが会話をコピーして生成AIへ貼り付ける必要がなく、Botが現在のチャンネルの文脈を直接取得します。また、整理した項目には根拠となったDiscordメッセージへのリンクが表示されるため、AIの要約が正しいかをすぐに確認できます。

## 生成AIだけに依存していない工夫

- Discord APIから指定件数のメッセージを取得する
- Bot投稿や空の投稿を除外する
- 投稿者を参加者番号へ匿名化する
- メールアドレス、電話番号、URLなどを伏せる
- GeminiへJSON形式で出力させる
- AIが返した根拠IDを検証する
- DiscordのEmbedに整理して表示する
- 根拠となる元メッセージへのリンクを付ける
- Render用のHTTPサーバーとヘルスチェックを用意する

## コマンド

- `/help`：Botの使い方を表示
- `/digest`：直近の会話を整理
- `/privacy`：匿名化とデータの扱いを表示

## 使用技術

- Node.js
- discord.js
- Gemini API（gemini-2.5-flash）
- Express
- Render

## 個人情報への対応

会話全文は保存しません。Geminiへ送信する前にDiscordの表示名とユーザーIDを参加者番号へ置き換え、メールアドレス、電話番号、URL、メンション、IPアドレス、長い機密文字列を伏せます。提出時の動作確認には架空の会話を使用します。
